Public API Methods you can Call

!!!  NOTICE!

Request URL Format:

INSERT_LINK_HERE/public_api/Mehod_Name?apikey=API_Key&...

GET  Counties

  • Description:

    Returns a County Array containing the Counties including ID and Name

  • In:

    No Input

  • Out:

    Array<County>

  • Example:

    No Example
GET  Cities

  • Description:

    Returns a City Array containing City ID, County ID and Name from a given County. If no County is specified, will contain all Cities

  • In:

    county (County ID - Optional)

  • Out:

    Array<City>

  • Example:

    No Example
GET  Institutions

  • Description:

    Returns a Institution Array containing ID and Name

  • In:

    city (CityID)

  • Out:

    Array<Institution>

  • Example:

    No Example
GET  Company

  • Description:

    Returns the Company of the specified CUI containing Name, CUI and Active Status in our System

  • In:

    cui (Company CUI)

  • Out:

    Company Object

  • Example:

    No Example
GET  CompanyByName

  • Description:

    Returns a Company Array from a given Name used as a query, containing Name, CUI and Active Status in our System

  • In:

    name (Name)

  • Out:

    Array<Company>

  • Example:

    No Example
GET  CompanyBranches

  • Description:

    Returns a Integer Array of the Company having the specified CUI, containing CityIDs where branches are present

  • In:

    cui (Company CUI)

  • Out:

    Array<Integer>

  • Example:

    No Example
GET  Tickets

  • Description:

    Returns a Ticket Array issued for the given BranchID, containing Name, Short Description, Long Description, Latitude (Optional) and Longitude (Optional)

  • In:

    branch (Branch ID)

  • Out:

    Array<Ticket>

  • Example:

    No Example
GET  Ticket

  • Description:

    Returns a Ticket Object from the given ID, containing Name, Short Description, Long Description, Latitude (Optional) and Longitude (Optional)

  • In:

    ticket (Ticket ID)

  • Out:

    Ticket Object

  • Example:

    No Example
