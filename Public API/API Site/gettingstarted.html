Getting Started Get Access to the API

1 Create an Account

First things first.
In order to use the Data you Provide, you need an Account. Don`t worry, any account will do.
Go to INSERT_LINK_HERE and create an Account.

2 Get an API Key

In order to use our Public API, you will need to tell us who you are... that is where the API Key comes in.
Every Request requires an API Key to be attached. Getting one is very simple though.
Go to your Account Settings INSERT_LINK_HERE and scroll to API. Create a New Project and a fresh new API Key will be prvided..

3 Use the API!

Now you have everything you need in order to launch requests!
Go to INSERT_LINK_HERE to see the Available Methods and try a few.
Don`t forget to use your API Key!
